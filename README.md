# Social-Media

* Social Media App
-> A Social media web application built with Nextjs and Laravel. This project was created to practise full-stack web development and understand how real social media platform work behind the scenes.

* Introduction
-> Hi, I'm a 17 year old student who has been learning web development for about a year. I have worked on small projects and spent a lot of time exploring different technologies. So, I wanted to challenge myself by building something that combines frontend development, backend APIs, authentication, databases and user interactions in single application.

* Features
1. User registration and login
2. Secure authentication
3. Create, edit and delete posts
4. Like and comment on posts
5. User profiles
6. Follow and Unfollow users
7. Image Uploads
8. Personalized feed
9. Responsive design

* Technologies Used
-> Frontend 
1. Nextjs
2. React
3. Tailwind CSS
4. Shadcn

-> Backend
1. Laravel
2. Laravel Sanctum
3. Laravel Socialite
4. MySQL

* Setup
-> Frontend
cd frontend
npm install
npm run dev

-> Backend 
cd backend
composer install
php artisan migrate
php artisan serve
